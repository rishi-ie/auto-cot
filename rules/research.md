# Rule: Research

## When to use
Task involves: finding information, literature review, fact-checking, source evaluation, data gathering, competitive analysis, or any "what is X?" or "what do we know about Y?" question.

## Trigger keywords
research, investigate, find, explore, analyze, compare, evaluate, review, survey, gather, collect, lookup, look up, find out, understand, learn about

## Reasoning steps

### 1. Define the question
State exactly what you need to know.
- "What is X?" vs "How does X work?" vs "Why did X happen?"
- Be specific — broad questions get broad, useless answers
- Break multi-part questions into sub-questions

### 2. Identify source types
What kinds of sources do you need?
- Primary sources (original data, documentation, direct statements)
- Secondary sources (analysis, synthesis, interpretation)
- Expert opinions (named authorities with credentials)
- General knowledge (widely accepted facts)

### 3. Search systematically
Find sources in order of reliability:
1. Official documentation / primary sources
2. Peer-reviewed research / academic papers
3. Expert-created content (reputable blogs, articles)
4. General web content (least reliable)

### 4. Evaluate each source
For each source, ask:
- Who created this? What are their credentials?
- When was it created? Is it current?
- Where was it published? Is the venue reputable?
- What is their agenda or perspective?
- Do they cite their sources?

### 5. Extract relevant information
Extract only what answers your question.
- Don't over-summarize — keep nuance
- Note the source for every fact
- Distinguish facts from interpretations
- Flag conflicting information

### 6. Synthesize
Combine information from multiple sources.
- Where do sources agree? That's likely solid.
- Where do they disagree? Note both positions.
- What patterns emerge across sources?
- What is still unknown or contested?

### 7. Cite properly
Document what came from where.
- Primary claim → source
- Supporting evidence → source
- Contradictory evidence → source and explanation
- Avoid: "research shows" without specifying who researched what

## Red flags
- Single source without corroboration
- Viral claims without primary source
- Correlation mistaken for causation
- Outdated information presented as current
- Source with clear agenda/conflict of interest

## Exit criteria
- [ ] Original question answered specifically
- [ ] Sources documented and evaluated
- [ ] Conflicting information acknowledged
- [ ] Confidence level stated (high/medium/low)
- [ ] Remaining unknowns identified
